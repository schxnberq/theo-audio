== Description ==

A comprehensive blog theme with a fresh, clean & elegant design. The theme is designed with the classic elegance to keep your reader more focused on the content & images. Our professional designers have kept the design simple yet eye-pleasing. You can customize most of the theme settings with live preview. Native WordPress customizer is used to make it easy for you to edit theme settings. We have added a customizable slider for featured posts and a cover section at the top for your blog introduction. The theme is lightweight with less than 40kb style.css  (you can minimize it further easily). It is fully responsive theme built on world's most loved framework for responsive websites, Bootstrap 4. You can use all latest WordPress page builder plugins like Site Origin, Elementor, Beaver Builder etc. This is a perfect WordPress theme for any kind of blog. Bani is a classic blog theme with RTL language support and is completely translation-ready. Our support team is always there to answer your queries. The theme will be regularly updated with new features. Check demo at http://bani.themesease.com. To know more about the theme, visit https://themes.salttechno.com/wordpress-theme/bani-free-blog-theme/.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
